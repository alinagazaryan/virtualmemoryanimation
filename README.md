# Анимации работы ОС с виртуальной памятью

Этот репозиторий содержит анимации, демонстрирующие работу операционной системы с виртуальной памятью, стеком и динамически выделяемой памятью. 

---

## Цель проекта

- Показать, как ОС управляет страницами виртуальной памяти.
- Продемонстрировать работу стека и динамической памяти (heap).
- Отобразить процесс выделения памяти через `new` и `malloc`.
- Показать взаимодействие приложения с системными вызовами ОС.
- Визуализировать алгоритмы поиска свободной памяти и закрепления страниц за процессом.
- Симулировать задержки на стороне ОС при выделении ресурсов.

---

## Особенности анимаций

- **Подсветка кода приложения** при вызове выделения памяти.
- **Пошаговая цепочка вызовов**: `new` → `malloc` → WinAPI → ОС.
- **Визуализация поиска свободной памяти** и алгоритмов размещения.
- **Выделение страниц памяти блоками** с демонстрацией избыточного выделения страниц.
- **Симуляция времени ожидания ОС** (таймеры, часики).
- **Поддержка нескольких версий анимации** — от базовой схемы до полной демонстрации.
